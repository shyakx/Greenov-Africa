# GREENOV LTD Website

A modern, responsive website built with React and Bootstrap for GREENOV LTD - a company focused on green and sustainable solutions.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI**: Clean and professional design using Bootstrap 5
- **React Router**: Multi-page navigation with React Router
- **Component-Based**: Modular React components for easy maintenance
- **Image Integration**: Uses provided design images for each section

## Pages

- **Home**: Landing page with hero section and key features
- **Who We Are**: Company information and mission
- **The Problem**: Environmental challenges and solutions
- **Our Services**: Service offerings and solutions
- **Core Values**: Company values and principles
- **Our Impact**: Statistics and achievements
- **Gallery**: Image gallery showcasing work
- **Contacts**: Contact information and contact form

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.
